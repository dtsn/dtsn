--- meta: _edit_lock: "1280405662" _edit_last: "2" _clicky_tweetpost: "" _clicky_goal: a:2:{s:2:"id";s:0:"";s:5:"value";s:0:"";} title: Link Prefetching published: true tags: - CSS/HTML layout: post type: post status: publish --- Link prefetching is a very little known HTML 5 tag which is currently only supported by Firefox. It utilizes browser idle time to download documents that the user might visit in the near future. For example: 
<link rel="prefetch" href="page2.html">
Once the user has loaded up that page the next page (page2.html) will being loading significantly decreasing the time it takes to visit the next page. It's a great idea really, and because it's gracefully degregating it means that users in FF will get the benefit while everyone else wouldn't see the difference. Refer to the Mozilla FAQ for more information. (via Keyboardy)
